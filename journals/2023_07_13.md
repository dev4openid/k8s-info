### Manual scheduling
- For e.g. of nginx: - Pod Definition
- ```yaml
  apiVersion: v1
  kind: Pod
  metadata:
    name: nginx
    labels:
      name: nginx
  spec:
    containers:
      - name: nginx
        image: nginx
        ports:
          - containerPort: 80
    node: node01			# simplest method, then delete and apply to create pod
  
  ```
-
- create a pod Binding definition
- ```yaml
  apiVersion: v1
  kind: Binding
  metadata:
    name: nginx
  
  target:
    apiVersion: v1
    kind: Node
    name: node01
  ```
